Instructions for Incantasia!

You will be shown a set of magical artifacts that are used to form an incantation. Beside that is a field of many artifacts that you must navigate to find the correct items. You must touch each item indicated to complete the ritual, in the correct order, in a timely manner. As you touch each correct item, it will indicate that you have touched the right one. If you touch an incorrect item, you must start again and complete the ritual before you run out of time. As you complete each ritual, it will move you along to the next ritual. Take careful note of the magical incantations as they relate to each object - later on you will need to select the objects based on the incantation alone!

Good luck young wizard!
